# Morsecode Trainer

## Overview
A trainer for absolute beginners in morsecode who want to lerne the morse alphabet.

## Preconditions

ITU Morsecode

### Conventions

JS Datatype of morsecode characters:

```javascript
var itu = {
        A: ".-",
        B: "-...",
        ...
    };
```



## Backend

Run the backend with

    mvn clean install
    mvn tomcat7:run

the site should run at localhost:9090
