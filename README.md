### Conventions

JS Datatype of morsecode characters:

```javascript
var x = {
        a: ".-",
        b: "-...",
        ...
    };
```



### Set up node.js Server

* Get [npm](https://www.npmjs.com/get-npm)
* Go to ***`angular-morsecode-trainer/`*** 
* Type `npm install`
* Type `npm start`
