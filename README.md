# Morsecode Trainer [![Build Status](https://travis-ci.org/Simiil/morsecode-trainer.svg?branch=master)](https://travis-ci.org/Simiil/morsecode-trainer)

## Backend

Runs on Tomcat.

Run the backend with

    mvn clean install
    mvn tomcat7:run

the site should run at localhost:9090
